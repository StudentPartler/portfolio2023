Software Development

This Java-based project features the creation, deletion, and management of objects which are serialized for later use. It illustrates the application of RMI in remote procedure calls between a server and a client with a GUI interface allows viewing, and hiding objects.

Connected Device

The goal of this project is to develop a simple java client app that monitors the temperature and humidity in a room and will be measured using sensors from both the Sensehat and the Raspberry Pi. The sensors data will then be collected on the app console (Mobile Phone) though an MQTT Broker.

Mobile App

Two distinct mobile app projects have been undertaken. The first project involves the development of a GPS locator, with the added feature of the ability to switch between different environmental themes. The second app, on the other hand, is a fully functional MP3 media player that offers core functionalities such as play, pause, stop, and skip to the next song.

Web Development

The Web Development project entails the creation of an SQL database that establishes a connection with the server through tools such as XAMPP. This allows for the manipulation of data on the website using PHP scripts for both making updates and retrieving information..
